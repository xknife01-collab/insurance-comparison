# 🛡️ 전 보험사 통합 비교 플랫폼: 마스터 작업 가이드 (Standard Playbook)

이 문서는 실손 의료비 보험 세팅의 성공 사례를 바탕으로 작성된 **표준 작업 지침서**입니다. 모든 보험 카테고리(암, 간병, 치아 등) 작업 시 아래 순서와 기준을 반드시 준수합니다.

---

## **1단계: 데이터 정제 및 전처리 (Data Cleaning)**
원본 엑셀(CSV) 데이터를 DB에 넣기 전, 데이터의 무결성을 확보합니다.
- **중복 제거:** 동일한 상품명과 담보가 중복으로 합산되지 않도록 `clean.py` 스크립트를 통해 필터링합니다.
- **상품명 표준화:** '(무)배당', '다이렉트' 등 불필요한 수식어를 정리하여 깔끔한 `display_name`을 생성합니다.
- **결측치 처리:** 보험료가 0원이거나 누락된 행은 제외하되, 필수 정보(보험사, 상품명)는 하나도 누락되지 않도록 합니다.

## **2단계: 데이터베이스 적재 (DB Ingestion)**
Supabase `medical_rates` 및 `medical_products` 테이블 구조에 맞춰 데이터를 삽입합니다.
- **성별 분리:** 남자(Col 6), 여자(Col 7) 보험료를 명확히 구분하여 저장합니다.
- **컬럼 맵핑:** `컬럼_추가_1` ~ `컬럼_추가_N`에 원본 엑셀의 모든 보장 내용을 텍스트 형태로 보존합니다.
- **카테고리 분류:** '실속형(일반)', '노후형', '유병자형' 등 상세 타입을 구분하여 태깅합니다.

## **3단계: 계산 엔진 고도화 (Calculation Engine)**
단순 데이터 노출이 아닌, 사용자 맞춤형 지능형 계산 로직을 적용합니다.
- **연령별 배수 적용 (Age Multiplier):** 40세 기준 데이터를 바탕으로 연령별 지수를 적용합니다.
    - 20대: 0.65 / 30대: 0.80 / 40대: 1.0 (기준) / 50대: 1.5 / 60대: 3.0 / 70대: 4.5
- **할인/할증 로직:** 4세대 실손의 '비급여 차등제'와 같이 이용량에 따른 5단계 할인/할증(5% 할인 ~ 300% 할증) 로직을 모든 관련 보험에 이식합니다.

## **4단계: 정밀 분석 리포트 구현 (Analysis Report)**
사용자가 '비교' 버튼을 눌렀을 때 나타나는 결과 화면의 품질을 극대화합니다.
- **상담 리포트:** "월 예상 절감액"을 가장 눈에 띄게 배치하여 전환의 가치를 증명합니다.
- **1:1 비교 분석표:** 'Stay(기존 유지)' vs 'Switch(교체 제안)' 테이블을 통해 보장 변화를 한눈에 대조합니다.
- **가성비 점수:** 보장 대비 보험료 효율을 100점 만점으로 시각화(Gauge 차트)합니다.

## **5단계: 보장 변화 시뮬레이션 (Simulation)**
- **Diet / Standard / Luxury:** 3가지 플랜을 제안하여 예산에 따른 보장 변화를 시뮬레이션합니다.
*   **슬라이더 연동:** 보험료 투자에 따른 혜택 변화(예: 사고 시 혜택 282% 상승)를 역동적인 애니메이션으로 구현합니다.

---

## **AI(Antigravity) 준수 사항**
1. **세션 시작 시:** 이 가이드를 먼저 읽고 현재 작업 중인 보험이 어느 단계인지 파악합니다.
2. **코드 수정 시:** ad-hoc(임시) 수정을 지양하고, 이 가이드에 정의된 **표준 클래스와 로직**을 따릅니다.
3. **디자인 품질:** 사용자 경험(UX)은 항상 '프리미엄' 수준을 유지하며, 시각적으로 WOW를 줄 수 있는 차트와 애니메이션을 활용합니다.
